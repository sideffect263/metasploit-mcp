Installation & Setup

Follow these steps to set up the Metasploit MCP Server:

# 1. Clone the repository
git clone https://github.com/your-repo/metasploit-mcp
cd metasploit-mcp

# 2. Install dependencies
pip3 install -r requirements.txt

# 3. Set environment variables
export MSF_PASSWORD='your-password'
export MSF_SERVER='127.0.0.1'
export MSF_PORT='55553'
export MSF_SSL='false'

# 4. Start Metasploit RPC daemon
msfrpcd -P your-password -S -a 127.0.0.1 -p 55553

# 5. Run the MCP server
python3 metasploit_mcp_server.py

Requirements:

  • Python 3.7 or higher
  • Metasploit Framework
  • pymetasploit3 library
  • MCP library

Connecting to the Server

Configure your MCP client to connect to this server:

# Claude Desktop Configuration (config.json)
{
  "mcpServers": {
    "metasploit": {
      "command": "python3",
      "args": ["/path/to/metasploit_mcp_server.py"],
      "env": {
        "MSF_PASSWORD": "your-password",
        "MSF_SERVER": "127.0.0.1",
        "MSF_PORT": "55553",
        "MSF_SSL": "false"
      }
    }
  }
}
# Python MCP Client Example
from mcp import ClientSession
from mcp.client.stdio import stdio_client

async def connect_to_metasploit():
    async with stdio_client(
        command="python3",
        args=["metasploit_mcp_server.py"]
    ) as (read, write):
        async with ClientSession(read, write) as session:
            # Initialize the connection
            await session.initialize()
            
            # List available tools
            tools = await session.list_tools()
            print(f"Available tools: {[tool.name for tool in tools.tools]}")
            
            # Call a tool
            result = await session.call_tool("list_exploits", {"search": "windows"})
            print(result)

Usage Examples

Common usage patterns and examples:

  "arguments": {

  }

  "tool": "get_exploit_info",
  "arguments": {

  }

# Example 3: Run a scanner
{
  "tool": "run_auxiliary",
  "arguments": {
    "module_name": "scanner/portscan/tcp",
    "options": {
      "RHOSTS": "192.168.1.0/24",
      "PORTS": "1-1000"
    }
  }
}

# Example 4: Start a handler
{
  "tool": "start_handler",
  "arguments": {
    "payload_type": "windows/meterpreter/reverse_tcp",
    "lhost": "192.168.1.100",
    "lport": 4444
  }
}

API Reference

Complete API documentation for all available tools:

# Environment Variables
MSF_PASSWORD   - Metasploit RPC password (default: 'yourpassword')
MSF_SERVER     - Metasploit RPC server (default: '127.0.0.1')
MSF_PORT       - Metasploit RPC port (default: '55553')
MSF_SSL        - Use SSL connection (default: 'false')

# Tool Response Format
{
  "status": "success|error",
  "data": {...},
  "error": "error message if applicable"
}

# Common Parameters
- search: String filter for searches
- limit: Maximum number of results
- options: Dictionary of module-specific options
- session_id: Target session identifier
- job_id: Background job identifier

Security Considerations:

  • Always use strong passwords for MSF RPC
  • Restrict network access to authorized hosts
  • Monitor server logs for suspicious activity
  • Use SSL/TLS in production environments
